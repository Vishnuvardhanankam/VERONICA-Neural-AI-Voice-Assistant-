# VERONICA-Neural-AI-Voice-Assistant-
VERONICA is an offline-capable AI voice assistant built with Python, Flask, JavaScript, and Ollama-powered Llama 3. It features voice interaction, wake-word detection, speech-to-text, text-to-speech, weather updates, fact-checking, recipe generation, note-taking, and a cyberpunk HUD interface while prioritizing privacy through local AI processing.
